# Pokémon Showdown Data Directory

The `data` directory contains essential data files that are used by various parts of the Pokémon Showdown project. These files include information about abilities, items, moves, Pokédex entries, and more. Below is a brief overview of each file and folder within the `data` directory.

## Main Files

- **abilities.ts**: Contains definitions for all Pokémon abilities.
- **aliases.ts**: Maps aliases to their corresponding names or IDs.
- **conditions.ts**: Defines various conditions that can affect battles, such as weather effects or status conditions.
- **formats-data.ts**: Provides data for different formats and rulesets used in the game.
- **FORMES.md**: Contains information about different forms of Pokémon.
- **items.ts**: Includes definitions for all items that can be used by Pokémon.
- **learnsets.ts**: Defines which moves Pokémon can learn at each level or through specific methods.
- **moves.ts**: Contains details about all available moves in the game.
- **natures.ts**: Lists the different natures that affect a Pokémon's stats.
- **pokedex.ts**: Provides comprehensive information about every Pokémon species.
- **pokemongo.ts**: Includes data specifically for Pokémon Go, which is not used in the core game but can be useful for mods and integrations.
- **rulesets.ts**: Defines rulesets for different formats, such as OU, Ubers, etc.
- **scripts.ts**: Contains various scripts that are used to process or manipulate data.
- **tags.ts**: Lists tags that can be applied to Pokémon, moves, items, etc., for filtering and organization purposes.
- **typechart.ts**: Defines the type chart, showing how different types interact with each other.

## Mod Folders

The `mods` folder contains additional data files for specific mods or customizations. Each subfolder represents a different mod:

- **chatbats/**: Contains data for chat-based mods that modify chat messages.
- **fullpotential/**: Data for the Full Potential mod, which adds new abilities and moves.
- **fusion_gen/**: Data for the Fusion Gen mod, which introduces fusion mechanics.
- **gen1/**: Data specific to Generation 1 Pokémon.
- **gen2/**: Data specific to Generation 2 Pokémon.
- **gen3/**: Data specific to Generation 3 Pokémon.
- **gen4/**: Data specific to Generation 4 Pokémon.
- **gen5/**: Data specific to Generation 5 Pokémon.
- **gen6/**: Data specific to Generation 6 Pokémon.
- **gen7/**: Data specific to Generation 7 Pokémon.
- **gen8/**: Data specific to Generation 8 Pokémon.
- **mixandmega/**: Data for the Mix and Mega mod, which adds new abilities and moves.

## Random Battles

The `random-battles` folder contains data specifically for random battles:

- **chatbats/**: Chat-based mods for random battles.
- **gen1/**: Generation 1 Pokémon data for random battles.
- **gen2/**: Generation 2 Pokémon data for random battles.
- **gen3/**: Generation 3 Pokémon data for random battles.
- **gen4/**: Generation 4 Pokémon data for random battles.
- **gen5/**: Generation 5 Pokémon data for random battles.

## Text Files

The `text` folder contains text-based data files:

- **abilities.ts**: Contains text descriptions of abilities.
- **default.ts**: Default text settings.
- **items.ts**: Text descriptions of items.
- **moves.ts**: Text descriptions of moves.
- **pokedex.ts**: Text descriptions of Pokédex entries.

## Summary

The `data` directory is crucial for the Pokémon Showdown project, providing essential data that powers various features and mechanics. Each file and folder serves a specific purpose, making it easier to understand and modify the game's data as needed.

For more detailed information on each file and its contents, refer to the individual files or consult the source code.
